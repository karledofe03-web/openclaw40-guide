# 环境准备

在开始安装 OpenClaw 之前，请确保你的系统满足以下要求。

## 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Linux | Ubuntu 22.04 |
| CPU | 2核 | 4核 |
| 内存 | 4GB | 8GB |
| 存储 | 20GB | 50GB SSD |

## 软件依赖

- Node.js v18+
- npm v9+
- Git
- Docker（推荐）

## 检查环境

```bash
node --version
npm --version
git --version
